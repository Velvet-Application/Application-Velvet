# ADR-0041 – Velvet Conversations Foundations

## Decisions validated

1. A Velvet Conversation is not a chat but the living space of a relationship.
2. Each conversation contains:
   - Shared Velvet Lounge
   - Private Velvet Notebook for each participant.
3. Conversations open on the latest unread activity, not necessarily the latest message.
4. Every relationship has a Timeline ('Histoire') showing key milestones while daily use stays focused on the conversation.
5. Participants may be added without creating a new conversation, only with unanimous consent. Existing members choose whether history is shared or the new participant starts from the invitation onward.

These principles become the architectural foundation of Velvet Conversations.
