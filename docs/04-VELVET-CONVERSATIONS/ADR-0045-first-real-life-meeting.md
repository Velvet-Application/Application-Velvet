# ADR-0045 - Première rencontre réelle

Décision validée.

Lorsqu'une rencontre physique a lieu lors d'un événement Velvet ou d'un événement partenaire permettant de confirmer les présences, chaque participant peut indiquer avoir rencontré un ou plusieurs profils.

Règles :
- aucune notification si une seule personne déclare la rencontre ;
- la rencontre n'est ajoutée à l'Histoire de la relation que si les deux parties la confirment ;
- une Invitation Velvet contextualisée peut être proposée après l'événement ;
- aucun détail intime n'est enregistré, uniquement le jalon 'Première rencontre réelle'.
