# ADR-0051 - Profils centrés sur la personne et galeries artistiques

Décision validée.

Sur un profil Velvet, la personne, le couple et leur univers prédominent. La photographie accompagne la découverte sans réduire le profil à une galerie de corps.

## Profil

Le profil met en avant :
- Une photographie principale
- L'identité et la philosophie du membre ou du couple
- Les valeurs, envies et centres d'intérêt
- Le Pacte Velvet
- L'Indice de Confiance
- Les événements, expériences et recommandations

Le message implicite est : `Découvrez qui nous sommes avant de découvrir nos photos.`

## Albums

Lorsqu'un album est ouvert, la photographie devient le contenu principal. L'expérience s'inspire d'une galerie ou d'un portfolio artistique :
- Images de grande taille
- Mise en page respirante
- Transitions sobres
- Interface secondaire discrète

## Réactions Velvet

Les photographies peuvent recevoir des réactions :
- J'aime
- Flamme
- Magnifique
- Élégant
- J'adore
- Inspirant

Les compteurs restent discrets. Les commentaires doivent être respectueux, bienveillants et soumis aux règles de modération Velvet.

Signature produit : le profil est un portrait ; l'album est une galerie.
